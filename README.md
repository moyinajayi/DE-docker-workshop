# DE-Zoomcamp-workshop

## Overview
This repository contains the code and resources for the Data Engineering Zoomcamp Workshop. The workshop focuses on building and orchestrating data pipelines using Docker, Google Cloud Platform (GCP), and Python. I will learn how to manage data ingestion, processing, and storage efficiently.

## Repository Structure

- **homeworks/**: Contains homework assignments for the workshop.


- **pipeline/**: Core pipeline scripts and configurations.
  - `Dockerfile`: Docker image definition for the project.
  - `docker-compose.yml`: Docker Compose configuration for managing services.
  - `ingest_data.py`: Script for ingesting data into the pipeline.
  - `pipeline.py`: Main script orchestrating the data pipeline.

- **test/**: Contains test files and scripts for validating the pipeline.

- **workflow-orchestration/**: Placeholder for workflow orchestration tasks.

## Prerequisites

- Docker and Docker Compose installed on your system.
- Python 3.8+ installed.
- Google Cloud Platform (GCP) account with access to BigQuery and GCS.
